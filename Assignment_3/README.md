# CSCI611 – Assignment 3
Traffic Sign Detection using YOLOv8

This assignment explores object detection using the YOLOv8 model on a custom traffic sign dataset.

Steps done:
- Downloaded dataset from Roboflow
- Evaluated pretrained YOLOv8n model
- Fine-tuned YOLOv8n for 30 epochs
- Compared performance metrics
- Experimented with different confidence and IoU thresholds

Results show significant improvement after fine-tuning, especially for small traffic sign detection.
